## üìã Overview

**GreenLine** is a modern, interactive bus ticket booking platform designed to revolutionize the way passengers reserve their seats. Built with a focus on user experience and functionality, this application provides a seamless journey from seat selection to booking confirmation. With an intuitive interface and real-time seat availability tracking, GreenLine makes bus travel booking effortless and enjoyable.

## üöÄ Features Breakdown

### Seat Booking System

- Visual seat grid with 40 available seats (A1-J4)
- Color-coded seat status (Gray: Available | Green: Selected)
- Maximum 4 seats per booking restriction
- Click to select/deselect functionality
- Automatic seat numbering and display

### Payment & Pricing

- Base fare: 550 Taka per seat
- Multiple coupon codes support
  - `NEW15` - 15% discount
  - `COUPLE20` - 20% discount
- Grand total calculation with discount application
- Clear price breakdown display

### Form Management

- Required field validation (Name, Phone, Email)
- Phone number format verification
- Email pattern validation
- Submit button activation based on booking state
- Success modal on successful submission

### Route Information

- Departure location: Dhaka
- Destination: Sylhet
- Departure time: 9:00 PM
- Estimated journey: 11 hours
- Multiple boarding points available
